# Shaj Theme System Showcase

The Shaj Theme System provides a comprehensive theming solution for the Atomix Design System, offering multiple pre-built themes and the ability to create custom variations.

## Available Themes

### Professional Themes

#### Shaj Default

A clean, modern theme perfect for professional applications with vibrant blue primary colors.

#### Shaj Midnight

A sophisticated dark theme with deep purple accents, ideal for professional night-time applications.

### Nature & Wellness Themes

#### Shaj Ocean

A calm, serene theme with aquatic colors perfect for wellness and healthcare applications.

#### Shaj Forest

A natural, organic theme with forest colors for environmental and wellness applications.

#### Shaj Pastel

A soft, gentle theme with pastel colors ideal for children's and wellness applications.

### Creative Themes

#### Shaj Sunset

A warm, energetic theme with sunset colors perfect for creative and lifestyle applications.

## Theme Features

- **Complete Color Palettes**: Each theme includes primary, secondary, neutral, and semantic color scales (50-900)
- **Typography System**: Consistent font families, weights, sizes, and line heights
- **Spacing Scale**: Comprehensive spacing system from 0 to 96 (384px)
- **Component Integration**: All components automatically adapt to theme colors and styles
- **CSS Variables**: Runtime theme switching without page reload
- **Accessibility**: WCAG 2.1 AA compliant with proper contrast ratios

## Usage

### Basic Theme Application

```html
<!-- Apply theme to entire page -->
<body data-theme="shaj-ocean">
  <!-- All components will use ocean theme -->
</body>

<!-- Apply theme to specific section -->
<div data-theme="shaj-sunset">
  <!-- Components in this section use sunset theme -->
</div>
```

### Runtime Theme Switching

```javascript
// Switch theme dynamically
document.body.setAttribute('data-theme', 'shaj-midnight');

// Or use the theme utility (if implemented)
window.Atomix.setTheme('shaj-forest');
```

### Using Theme Variables in CSS

```scss
.my-component {
  background-color: var(--shaj-primary-500);
  color: var(--shaj-neutral-50);
  padding: var(--shaj-spacing-4);
  border-radius: var(--shaj-border-radius-md);
  box-shadow: var(--shaj-shadow-md);
}
```

## Theme Customization

The Shaj Theme System includes a powerful variant generator for creating custom themes:

```scss
// Create a custom variant
$my-custom-theme: variant.create-theme-variant(
  $base-theme,
  (
    colors: (
      primary: (
        hue-shift: 30deg,
        saturation: 10%,
      ),
      secondary: (
        replace: (
          500: #custom-color,
        ),
      ),
    ),
    spacing: 1.2,
    // Scale spacing by 20%
    components: (
        button: (
          border-radius: 1rem,
          font-weight: 600,
        ),
      ),
  )
);
```

## Best Practices

1. **Consistent Theme Application**: Apply themes at the highest level possible for consistency
2. **Semantic Colors**: Use semantic color variables (success, warning, error) for better maintainability
3. **Component Variants**: Leverage theme-aware component variants for better visual hierarchy
4. **Accessibility**: Test themes with screen readers and ensure proper contrast ratios
5. **Performance**: Use CSS custom properties for runtime switching without performance impact

## Browser Support

- Modern browsers with CSS custom properties support
- Graceful fallback for older browsers through SCSS compilation
- Progressive enhancement approach ensures basic functionality everywhere
